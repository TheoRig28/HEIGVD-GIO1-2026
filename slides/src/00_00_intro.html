Introduction

Le projet

Le projet devra

  • Se faire par groupe de 2 ou 3 personnes
  • Se présenter sous la forme d'une application web, utilisant HTML, CSS et JavaScript
  • Utiliser la librairie VueJS
  • Utiliser au moins une librairie de mapping 2D (Leaflet, Openlayers, ou autre)
  • Utiliser au moins une librairie de mapping 3D (CesiumJS, Giro3D, ou autre)
  • Présenter des données issues d'une API externe (par exemple https://docs.geo.admin.ch/ ou similaire)
  • Utiliser git pour la gestion du code et être hébergé sur GitHub

Le projet est à rendu via le dépôt GitHub, la présentation du projet sera de 15mins

Date de rendu : le 16.04.2026 à 23h59 sous réserve d'être accepté par la HES-SO
