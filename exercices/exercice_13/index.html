Exercice 13

Note : Cet exercice utilise les modules JavaScript. Pour que ceux-ci fonctionnent en local sur votre ordinateur, il vous faut un serveur web local. Vous pouvez démarrer un serveur web local en ouvrant un terminal et en vous plaçant dans le dossier des exercices, puis en tapant la commande : python3 -m http.server ou python -m http.server.
Dans votre navigateur, allez ensuite à la page 127.0.0.1:8000

En relisant l'exercice 12, vous trouvez cette fois que la classe Polygon peut être améliorée.
En reprenant le code de l'exercice précédent, réécrivez les fonctions de la classe Polygon de manière à n'utiliser que des variables immuables et en vous servant des méthodes des tableaux JavaScript.
Les méthodes de la classe Polygon ne devront contenir aucune variable (seules les constantes sont autorisées), et ne contenir aucune boucle explicite (pas de boucle for ni while).

Conseil : Regardez les méthodes suivantes de la classe Array :

  • Array.reduce()
  • Array.map()
