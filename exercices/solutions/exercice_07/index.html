Exercice 07

Note : Cet exercice utilise les modules JavaScript. Pour que ceux-ci fonctionnent en local sur votre ordinateur, il vous faut un serveur web local. Vous pouvez démarrer un serveur web local en ouvrant un terminal et en vous plaçant dans le dossier des exercices, puis en tapant la commande : python3 -m http.server ou python -m http.server.
Dans votre navigateur, allez ensuite à la page 127.0.0.1:8000

On vous donne les classes Point et Polygon de l'exercice 5, complétées.
Sans modifier le code de ces classes, créez un module "ProxyLogger" qui, de la manière décrite dans le fichier main.js, créé un proxy qui affiche un message dans la console à chaque :

  • Lecture d'un attribut
  • Exécution d'une méthode
  • Ecriture d'un attribut

Le comportement des classes ne doit pas être changé.
Créez un seul module ProxyLogger, identique pour les deux classes, comme utilisé dans le fichier main.js


Avec un Proxy, l'appel à une fonction est traité dans le handler get. Vous pouvez vérifier si un objet est une fonction avec l'instruction
if(typeof obj === "function"){...}
